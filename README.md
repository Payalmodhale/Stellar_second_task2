# 🚀 Multi Wallet Payment Tracker (Level 2)

## 📌 Description
A Stellar blockchain project with multi-wallet support, balance checking, XLM transfer, transaction status, and error handling.

## ✨ Features
- Create multiple wallets
- Switch wallets
- Check balance
- Send XLM
- Live transaction status
- Error handling

## ⚙️ Run
npm install
npm run dev
